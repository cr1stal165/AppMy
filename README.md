# Мобильное приложение ChopTalk
---
ChopTalk - аналог чата со следующими возможностями:
---
1. Отправка сообщений
2. Отправка emoji
3. Просмотр профиля
4. Авторизация
5. Регистрация

---
# Стэк технологий:
1. ЯП - Java
2. БД - Firebase
3. API - worldtimeapi
